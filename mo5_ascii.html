ASCII commands and special chars

Sometimes ASCII is a good way to set up the screen or palette without too much code...

  • 0 NUL
  • 1 SOH
  • 2 STOP KEY (basic)
  • 3 BREAK (basic)
  • 4 EOT
  • 5 ENQ
  • 6 ACK
  • 7 beep
  • 8 cursor left
  • 9 cursor right
  • A cursor down
  • B cursor up
  • C RAZ (clear screen)
  • D return (begin of next line)
  • E teletel semigraphic mode
  • F alphanum mode
  • 10 DLE
  • 11 show cursor
  • 12 nothing
  • 13 nothing
  • 14 Hide cursor
  • 15
  • 16 ACC key (call G2 graphics)
  • 17
  • 18 erase to EOL
  • 19
  • 1A
  • 1B Escape (see below)
  • 1C insert (basic)
  • 1D erase (basic)
  • 1E tab (basic)
  • 1F US (set cursor or window, see below)

Escape and US commands

  • 1B 20 4x Select frontcolor for whole screen
  • 1B 20 5x Select backcolor for whole screen
  • 1B 4x Select frontcolor
  • 1B 5x select backcolor
  • 1B 6x Select border color
  • 1B 70 Normal size chars
  • 1B 71 Double height
  • 1B 72 Double width
  • 1B 73 Double size
  • 1B 74 Opaque chars (use color ram)
  • 1B 75 Transparent chars (don't touch the color RAM)
  • 1B 76 video incrustation
  • 1B 78 fast scroll
  • 1B 79 Soft scroll
  • 1B 7A page mode (no scroll)
  • 1B 7B Video invert (negate pixeldata)
  • 1F 2n 2n Window top border to line nn
  • 1F 1n 1n Window bottom border to line nn
  • 1F 4n 4n Move cursor (n may be > 0xF, for example 1F 48 50)

Teletel mode allows you to use semigraphic chars. Switch on by sending 0xE, off with 0xF. In Teletel mode each char is divided in 6 areas as follow:

AB
CD
EF

To print a char, encode its data as 0F1EDCBA. Other values may not be used. Control chars could still work (?) and colors will be taken into account (?).
