GrafX2

Code: PulkoMandy

GrafX2 is an old pixel painting program from the DOS era. The original authors, Sunset Design, stopped developping it and released the sourcecode into the wild. We did a port to linux and windows and we are now adding some features to the program.

Will run on Linux, Windows, and somewhat on AmigaOS 4 and Mac OS X. May be compiled on any platform with SDL available.

MO5 SDK

Code : PulkoMandy & Cloudstrife

This is a set of tools for developping on MO5. It only fills the missing parts between other tools. So far it contains two tools :

  • gfx2mo5 : converts png pictures to the mo5 fileformat.
  • saptofd : converts sap image file (from sapfs tool for example) to fd format (for dcmo5)
